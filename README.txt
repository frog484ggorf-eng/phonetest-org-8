Инструкция по развёртыванию на Netlify
====================================

1. Распакуйте архив и откройте содержимое.
2. Вы можете просто перетащить папку с файлами (index.html, style.css, script.js) в Netlify Drop:
   https://app.netlify.com/drop
   (или развернуть через репозиторий GitHub — Netlify автоматически деплоит статические сайты).
3. Файлы:
   - index.html — главная страница
   - style.css — стили
   - script.js — логика теста
4. Нет бэкенда — всё работает статически в браузере.
5. При необходимости можно изменить вопросы в script.js в массиве QUESTIONS.

Удачного деплоя!
